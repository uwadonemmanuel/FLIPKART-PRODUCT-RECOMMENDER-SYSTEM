INSTALLATION INSTRUCTIONS:
--------------------------
1. Open package_downloads.html in your browser for a user-friendly interface
2. Or use the URLs in this file to download packages manually
3. Download the .whl files for macOS x86_64 (cp312-cp312-macosx_10_13_x86_64)
4. Save all downloaded .whl files in a folder (e.g., packages/)
5. Install them using: pip install packages/*.whl
6. Or install individually: pip install /path/to/package.whl

NOTE: Make sure to download the correct wheel file for your platform:
- macOS x86_64: Look for files with "macosx_10_13_x86_64" or "macosx_10_12_x86_64"
- Universal wheels (.whl files without platform-specific tags) work on all platforms
